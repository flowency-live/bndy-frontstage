Implementation Notes

  • Glow achieved via stacked ScatterplotLayers with decreasing opacity — no PostProcessEffect needed
  • Breathing animation uses requestAnimationFrame + updateTriggers for GPU-accelerated interpolation
  • "Tonight" sonar effect uses animated stroke radius that expands and fades
  • Labels use TextLayer with outlineWidth for halo effect — fully GPU-rendered
  • Integration via MapboxOverlay with interleaved: true — shares Mapbox's WebGL context
  • Bundle size: +150KB gzipped (replaces HTML marker system overhead)
